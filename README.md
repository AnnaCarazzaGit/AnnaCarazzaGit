### About me

My works involve **wearable biomedical sensing, embedded systems, and signal processing** — from designing the hardware and firmware that acquire a physiological signal, to the pipelines that clean, analyze, and extract meaning from it.

I'm interested in physiological signal processing, embedded biosensor design, and machine learning applied to biomedical data.

### 🔬 Main projects

- **[lteb_PAT_project](https://github.com/AnnaCarazzaGit/lteb_PAT_project)** — Wearable blood pressure monitor
  Hardware, firmware, and software design of a smart wearable device estimating blood pressure via Pulse Arrival Time, built on the nRF52840 SoC (ECG + PPG acquisition, BLE sync, custom Streamlit app for real-time monitoring).

- **[nn_fMRI_project](https://github.com/AnnaCarazzaGit/nn_fMRI_project)** — Neural decoding via deep learning
  Hierarchical deep learning pipeline for classifying fMRI brain activity data.

- **[ss_homeworks](https://github.com/AnnaCarazzaGit/ss_homeworks)** — Embedded biosensor systems
  A collection of STM32 (NUCLEO-F401RE) and Arduino projects covering GPIO/interrupts, ADC, I2C/SPI/UART-USART sensor interfacing.

- **[ehma_SeriousGame_project](https://github.com/AnnaCarazzaGit/ehma_SeriousGame_project)** — Medical serious game
  A clinical serious game built in Unity, applying machine learning and clustering to analyze user behavioral data.

- **[mi_sleepDisordersManagementApp_project](https://github.com/AnnaCarazzaGit/mi_sleepDisordersManagementApp_project)** — Healthcare management app
  Software with a custom GUI for interactive doctor–patient data management.
